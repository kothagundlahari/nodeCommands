# nodeCommands

This repo illustrates the functionality of few node commands such as:
 
    1. diff
    2. ls
    3. find
    4. sed

# Installation
1. Clone or Download the repo. 
2. Run npm install 
    
    
    npm install 
    

# diff 

    node testDiffCommands.js file.txt file1.txt 

    
# ls

     node ls.js 
     
           
# find

      node find.js 
      
# sed
       node sed.js
